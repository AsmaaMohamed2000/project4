# HomeStyle E-Commerce Project

## Overview
HomeStyle is a simple *front-end e-commerce web application* built with *HTML, CSS, JavaScript, and Bootstrap*.  
It allows users to *register, login, browse products, add products to a shopping cart, and save favorite items*.  

This project uses *LocalStorage* to store user information, cart items, and favorites .

## Features

### User Management
- *Register:* Users can sign up by providing first name, second name, email, and password.  
- *Login:* Users can log in using their email and password.  
- *Logout:* Clears user data and resets the interface.  
- User interface updates dynamically based on login status.

### Product Catalog
- Products are displayed with:
  - Image
  - Title
  - Price
  - Category
- Users can *filter products by name or category* using the search bar.  
- Responsive layout using *Bootstrap grid*.

### Shopping Cart
- Add/remove products to/from the cart.  
- Adjust product quantity (+ / - buttons).  
- Displays *total price*.  
- Dynamic update of the cart icon count.  

### Favorites
- Mark/unmark products as favorites using the heart icon.  
- Favorites are displayed separately.  
- Favorites are stored in *LocalStorage* per user.

### Responsive Design
- Fully responsive for *desktop, tablet, and mobile*.  
- Uses *Bootstrap* and custom CSS for layout and styling.

## How to Use

1. Clone the Repository

```bash
git clone [https://github.com/AsmaaMohamed2000/project4]
```

2. Open index.html in browser 
3. Register a new account or login with an existing account.

4. Browse products, add to cart, or mark as favorite.

5. View cart by clicking the cart icon.

6. Logout to clear the session.
